# Harry Nguyen — Portfolio

Personal portfolio site showcasing my projects and skills as a full-stack developer.

**Live site:** [harry-nguyen.dev](https://harry-nguyen.dev/)

## Tech Stack

- **Framework:** Next.js 15 (App Router) + TypeScript
- **Styling:** Tailwind CSS 4
- **CMS:** Payload CMS
- **Database:** MongoDB (production) / SQLite (local dev)
- **Storage:** Cloudflare R2
- **Animation:** Motion, PixiJS
- **Deployment:** Railway

## Features

- Projects gallery with tag-based filtering and animated transitions
- Interactive PixiJS WebGL canvas background
- Rich text project detail pages via Payload's Lexical editor
- Responsive design with mobile nav
- Payload admin panel for content management
